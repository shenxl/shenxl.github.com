随着学习的深入，了解到了CentOS上安装软件的基本方式，也了解到了RPMForge仓库的知识:

RPMForge仓库是一个用于在Red Hat Enterprise Linux (RHEL) 和Community ENTerprise Operating System (CentOS)中安装第三方软件包的工具。它提供了超过5000个rpm格式的软件包。
RPMForge本身并不属于RHEL或CentOS系统的一部分，不过其设计意图即是为这些系统服务。可以在 http://packages.sw.be/ 浏览所有的RPMForge包列表。

安装Scrot截屏工具

参考文档

  1. 使用刚刚新安装的RPM 仓库安装giblib-devel，这是Scrot的依赖库
    yum install giblib-devel

  2. 下载并安装Scrot

  3. 当make的时候，发现shell抛出error:

    gcc的编译参数为：

    搜索网上的资源了解到：
    gcc 编译库时的简写规则为： l + 库名 => lib + 库名 + . + so
    所以 lfreetype 在库中表现即为：libfreetype.so
    查找目录/usr/lib64 注意，64位版本的Lib目录为lib64! 发现编译的目标类库分别命名为：
    libfreetype.so.6 、 libXext.so.6 。找到问题的原因后，使用ln -s命令建立编译的软链接即可：

  4. 至此，截屏软件已经可以使用了。具体的使用方式可以参考:scrot 从入门到精通
